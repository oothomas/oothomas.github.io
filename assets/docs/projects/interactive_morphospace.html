Morphospace Viewer

How to Use This Tool

  1. Fetch PCA Data: Click “Fetch PCA Data” to load up to 15 principal components from the server. The chart will display all specimens, color-coded by group.
  2. Select PCs: Use the “X PC” and “Y PC” dropdowns to choose which two principal components to view on the scatter plot. Changing them clears any existing picks or shapes.
  3. Pick Source & Target:
    • Click “Set Source Point” and then single-click anywhere in the chart. That location becomes the source (shown in green).
    • Click “Set Target Point” and single-click in the chart for the target (shown in blue).
    A dashed gray line indicates the interpolation path between source and target.
  4. Compute & Interpolate: Once you have both a source and target, the “Compute & Interpolate” button becomes active. Click it to send the zeroed PC array (with only those 2D coords set) to the server.
  5. View 3D Shapes:
    • The server returns a series of N shapes (aligned). We color them by per-vertex error relative to shape #0.
    • Use the slider below the scatter plot to step through the shapes [0..N-1].
    • The 3D viewer is 800px wide, matching the chart’s width for a cleaner layout.
  6. Repeat:
    • You can change the X/Y PCs at any time (this clears shapes and picks). Then pick a new source/target and run “Compute & Interpolate” again.
    • There is only one VTK view reused each time, so you won’t see multiple 3D windows stack up.
